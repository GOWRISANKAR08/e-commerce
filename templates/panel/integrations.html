Fast2SMS setup

1. Register at fast2sms.com and top up your wallet (₹10 minimum)

2. Go to Dev API → API Key and copy your key

3. Paste the API key below — no Account SID needed

4. Sender ID is optional (leave blank to use Fast2SMS default)

MSG91 setup

1. Register at msg91.com and recharge your account

2. Copy your Auth Key from the dashboard — paste it as API Key

3. Create an OTP template; note the Template ID and set a 6-digit OTP variable

4. Set your Sender ID (6-character DLT-registered sender ID)

Twilio setup

1. Sign up at twilio.com and get a phone number

2. From the console, copy your Account SID and Auth Token

3. Paste Auth Token as API Key / Auth Token and Account SID in its field

4. Set Sender ID to your Twilio phone number (e.g. +919876543210)

How image storage works

  • When Cloudinary is enabled, product images are uploaded to your Cloudinary account and only the URL is stored in the database.
  • When Cloudinary is disabled, images fall back to local MEDIA_ROOT storage on this server.
  • Free plan includes 25 GB storage + 25 GB bandwidth/month — no credit card required ever.
